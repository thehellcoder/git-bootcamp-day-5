# wordpress-deploy

Учебный Ansible-плейбук для курса Git Bootcamp (день 5).

> **Дисклеймер.** В реальном проекте такой монолитный `site.yml` разбили бы на 3–4 роли
> (`mysql`, `nginx`, `php-fpm`, `wordpress`). Здесь всё в одном файле — чтобы история
> коммитов была наглядной для упражнений с `git log`, `blame` и `revert`.
